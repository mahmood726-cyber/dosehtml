📖 Interpretation Guide

Egger's Test: Tests for funnel plot asymmetry.

  • p < 0.10: Suggests potential publication bias
  • p ≥ 0.10: No strong evidence of asymmetry
  • ⚠ Low power with <10 studies

Begg's Test: Rank correlation test for asymmetry.

  • Kendall's τ near 0: Symmetric
  • p < 0.10: Suggests bias

Trim & Fill: Estimates missing studies.

  • Shows adjusted estimate if studies imputed
  • ⚠ Assumes asymmetry = publication bias

⚠️ Important Caveats

Funnel plot asymmetry ≠ Publication bias. Other causes include:

  • True heterogeneity (different effects in different populations)
  • Methodological differences (smaller studies may use different methods)
  • Chance (especially with <10 studies)
  • Outcome reporting bias (within studies)

Reference: Sterne JA, et al. BMJ. 2011;343:d4002.

📐 Statistical Methods Appendix

Complete mathematical derivations and formulas for all statistical methods implemented in this tool.

1. Greenland-Longnecker Two-Stage GLS Method

Stage 1: Within-Study Covariance

For case-control studies with reference category counts available:

Cov(log RRj, log RRk) = 1/n0

Where n0 is the reference category case count.


When case counts are unavailable (approximation):

Cov(log RRj, log RRk) ≈ Var(log RRj) + Var(log RRk)


Stage 2: GLS Estimation

β̂ = (X'V-1X)-1 X'V-1y

Var(β̂) = (X'V-1X)-1

Where V is the block-diagonal covariance matrix incorporating τ²:

V* = V + τ²I

2. REML Variance Estimation

REML Log-Likelihood:

ℓR(τ²) = -½[log|V*| + log|X'V*-1X| + (y-Xβ̂)'V*-1(y-Xβ̂)]


Fisher Scoring Update:

τ²new = τ²old + U(τ²)/I(τ²)

Where U is the score function and I is the Fisher information.


Score Function:

U(τ²) = ½[r'V*-1V*-1r - tr(V*-1)]

Where r = y - Xβ̂ is the residual vector.

3. Heterogeneity Statistics

Cochran's Q:

Q = (y - Xβ̂)'V*-1(y - Xβ̂)


Degrees of Freedom (Multivariate):

df = (K - 1) × p

Where K = number of studies, p = number of parameters.


I² Statistic:

I² = max(0, 100 × (Q - df) / Q)


Prediction Interval:

β̂ ± tdf,α/2 × √(SE²(β̂) + τ²)

4. Hartung-Knapp-Sidik-Jonkman Adjustment

Adjusted Variance:

VarHKSJ(β̂) = q × Var(β̂)

q = max(1, Q / (K - 1))


Confidence Interval:

β̂ ± tK-1,α/2 × √(VarHKSJ(β̂))

Uses t-distribution instead of normal for better small-sample coverage.

5. Restricted Cubic Splines

Spline Basis Function (Durrleman & Simon 1989):

Sk(d) = (d - tk)3+ - λk(d - tK)3+ + (1-λk)(d - t1)3+

λk = (tK - tk) / (tK - t1)

Where (x)+ = max(0, x) and t1...tK are knot locations.

6. Bootstrap Confidence Intervals

Cluster Bootstrap (Study-Level Resampling):

For b = 1, ..., B bootstrap replications:

1. Resample K studies with replacement

2. Fit model to get β̂*b


Percentile CI:

[β̂*(α/2), β̂*(1-α/2)]


BCa CI (Bias-Corrected Accelerated):

z0 = Φ-1(#{β̂*b < β̂} / B) (bias correction)

a = Σ(β̂-i - β̂(.))³ / (6[Σ(β̂-i - β̂(.))²]3/2) (acceleration)

α1 = Φ(z0 + (z0 + zα/2)/(1 - a(z0 + zα/2)))

α2 = Φ(z0 + (z0 + z1-α/2)/(1 - a(z0 + z1-α/2)))
