Help & Documentation

📖 How to Use

1. Data Tab: Enter your dose-response data in CSV format. Each row represents a dose category within a study.
2. Settings Tab: Choose your model type, correlation, and analysis options.
3. Run Analysis: Click "Run Analysis" to fit the model.
4. View Results: Check the Results, Plots, and other tabs for outputs.

📊 Data Format

Required columns: id, dose, logRR (or RR), se
Optional columns: author, type, cases, n
Reference dose should have logRR = 0

🔬 Model Types

Linear: Straight line relationship
Quadratic: Parabolic curve (U-shaped or inverted U)
Cubic: S-shaped curve
Restricted Cubic Splines: Flexible smooth curve with knots
Fractional Polynomials: Power transformations
GLS: Greenland-Longnecker method with covariance matrix

📈 Interpreting Results

Q Statistic: Test for heterogeneity (p < 0.05 indicates significant heterogeneity)
I²: Percentage of variation due to heterogeneity (0-100%)
τ²: Between-study variance
95% CI: Confidence interval for coefficients
