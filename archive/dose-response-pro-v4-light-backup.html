📊 Study Data Entry

Enter dose-response data for each study. For each dose level, specify the dose, cases/events, and total sample size (or person-time).

📄 CSV Format Guide

Upload a CSV file with the following columns:

Study,Dose,Cases,N
Study1,0,10,100
Study1,10,15,100
Study1,20,25,100
Study2,0,8,100
Study2,10,12,100

Note: First row must contain headers. Multiple dose points per study are supported. Columns can be in any order as long as they contain: Study, Dose, Cases, N (or PersonTime).

How to Interpret

  • AIC (Akaike Information Criterion): Lower values indicate better fit while penalizing model complexity
  • BIC (Bayesian Information Criterion): Lower values indicate better fit with stronger penalty for complexity
  • Log-Likelihood: Higher values indicate better fit
  • I²: Percentage of variability due to heterogeneity between studies
  • τ²: Between-study variance (heterogeneity)
  • p-Trend: Statistical significance of the dose-response trend
