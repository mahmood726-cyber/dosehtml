================================================================================
METHOD DETAILS
================================================================================

The One-Stage Random-Effects Model:

1. ADVANTAGES:
   - Jointly fits all study data in a single model
   - More efficient than two-stage approaches when few studies
   - Properly accounts for within-study and between-study variance
   - Uses REML (Restricted Maximum Likelihood) for variance estimation

2. ALGORITHM:
   a) Collect all dose-response data points across studies
   b) Initialize with pooled OLS estimation
   c) Iteratively estimate tau² using REML-style method of moments
   d) Fit weighted least squares with converged tau²
   e) Calculate variance-covariance matrix for fixed effects
   f) Compute predictions, residuals, and model diagnostics

3. OUTPUT STRUCTURE:
   {
     type: 'onestage',
     name: 'One-Stage Random-Effects',
     description: 'Joint model fitting all study data simultaneously',
     coefficients: [Intercept, Linear, Quadratic],
     pTrend, pNonlinear, Q, I2, tau2,
     nStudies, nPoints, AIC, BIC, logLik,
     residuals: [...],
     predict: function(dose) { ... }
   }
